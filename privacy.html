Privacy Policy

Effective Date: Today

Overview

Ned is a completely offline, client-side browser extension designed for privacy and speed.

Data Collection

This extension collects zero personal data, telemetry, or analytics. It makes absolutely no external API calls to any servers. All CVSS calculations are processed locally within your browser using embedded JavaScript libraries.

Permissions

The extension requires minimal permissions (like modifying clipboard data briefly when you click to copy a vector string). It does not track your browsing history or access your web pages.

Open Source

You can verify this by viewing the open-source code on GitHub.
